## Outline User Roles

[👉🏾👉🏾👉🏾 Include at least at least 1, but no more than 3, user roles.]
- User
- Admin

## Draft User Stories

[👉🏾👉🏾👉🏾 Include at least at least 10 user stories in this format:]

1. As a [user role], I want to [what], so that [why].
1. As a user, I want to be able to add songs to my favorites list, so that I can create a list of songs.
1. As a user, I want to be able to add artists to my favorites list, so that I can create a comprehensive list of artists that I enjoy.
1. As a user, I want to be able to add genres to my favorites list, so that I can show what kind of music genres I like.
1. As a user, I want to be able to edit the songs that I have added to my list, so that I can correct mistakes.
1. As a user, I want to be able to edit the artist names, so that I can correct mistakes.
1. As a user, I want to be able to edit the genre names, so that I can correct mistakes.
1. As a user, I want to be able to delete songs, so that I can remove them from my favorites list.
1. As a user, I want to be able to delete artists, so that I can remove them from my favorites list.
1. As a user, I want to be able to delete genres, so that I can remove them from my favorites list.
1. As a user, I want to be able to delete my account, so that I can remove my account from the wbesite. 
1. As an admin, I want to be able to delete other users accounts, so that I can keep our website and database clean.
